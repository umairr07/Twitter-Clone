Welcome to Twiiter(X) Clone

API Documentation

Note: To use the APIs please signup and login first...

* User Authentication APIs *

Signup : https://twitter-clone-qtw8.onrender.com/api/v1/auth/signup (POST API)

Login : https://twitter-clone-qtw8.onrender.com/api/v1/auth/login (POST API)

Logout : https://twitter-clone-qtw8.onrender.com/api/v1/auth/logout/:id (POST API)


* User APIs *

User Profile : https://twitter-clone-qtw8.onrender.com/api/v1/users/profile/:username (GET API)

Suggested Users : https://twitter-clone-qtw8.onrender.com/api/v1/users/suggested (GET API)

Follow/Unfollow Users : https://twitter-clone-qtw8.onrender.com/api/v1/users/profile/:id (POST API)

Update Profile : https://twitter-clone-qtw8.onrender.com/api/v1/users/update (POST API)


* Post/Tweets APIs *

Create Post : https://twitter-clone-qtw8.onrender.com/api/v1/posts/create (POST API)

Delete Post : https://twitter-clone-qtw8.onrender.com/api/v1/posts/:id (DELETE API)

Like/Unlike Post : https://twitter-clone-qtw8.onrender.com/api/v1/posts/like/:id (POST API)

Comment on Post : https://twitter-clone-qtw8.onrender.com/api/v1/posts/comment/:id (POST API)

Get All Posts : https://twitter-clone-qtw8.onrender.com/api/v1/posts/all (GET API)

Get User Posts : https://twitter-clone-qtw8.onrender.com/api/v1/posts/userPosts/:username (GET API)

Get Following Posts : https://twitter-clone-qtw8.onrender.com/api/v1/posts/followingPosts (GET API)


* Notification Apis *

All Notifications : https://twitter-clone-qtw8.onrender.com/api/v1/notifications/all (GET API)

Delete All Notifications : https://twitter-clone-qtw8.onrender.com/api/v1/notifications/delete (DELETE API)
